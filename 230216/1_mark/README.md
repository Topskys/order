# 230216 Mark 
Mark is a desktop application for editing markdown.

## 技术选型
Electron + Vue.js + Node.js + MongoDB

## 启动项目

```bash
# 安装依赖
npm install
# 运行项目
npm run electron:serve
# 打包项目
npm run electron:build
```
